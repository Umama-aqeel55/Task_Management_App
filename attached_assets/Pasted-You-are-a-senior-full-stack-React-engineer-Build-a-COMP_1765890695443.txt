CORE FEATURES
=====================

BOARDS & TASKS
- Create boards
- Create columns (Todo, In Progress, Done)
- Create tasks inside columns
- Edit / delete tasks
- Assign task title, description, due date, priority
- Drag & drop tasks between columns
- Persist order in Firestore

TEAM COLLABORATION
- Invite members via email
- Assign tasks to users
- Display assigned user avatar
- Role: Owner / Member

REAL-TIME UPDATES
- Use Firestore real-time listeners
- Tasks move live for all users
- Board updates reflect instantly

PROGRESS TRACKING
- Progress bar per board
- Task completion percentage
- Charts for completed vs pending tasks

FILE UPLOAD
- Upload files to tasks
- Store files in Firebase Storage
- Attach file URLs to tasks
